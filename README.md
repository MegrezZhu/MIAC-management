# MIAC-management

## HOW TO

```bash
$ npm install # install dependencies
$ sudo npm install eslint -g # install global eslint for linting
$ npm run lint # lint your JavaScript code
  # front end
$ gulp # automatically build
  # back end
$ npm run dev # development mode
$ npm run build # compile the code under the `src` directory using babel
$ npm run serve # run the compiled code
```
